# Leap
Leap is a lightweight search engine developed by Lambda Core

## Features
- Blocking AI and NSFW results as per user settings
- Minimal impact on the environment
- Lightweight and capable of offline usage
- Adaptable for different uses

## Run Locally

Clone Leap
```bash
  git clone https://github.com/midnightdodger/Leap.git
```
Go to Leaps directory
```bash
  cd leap
```

## Install dependencies
## Arch based distros
```bash
  yay -S python3
```

## Debian based distros
```bash
  sudo apt install python3
```

## RedHat based distros
```bash
  sudo yum install -y python34
```

## Windows/MacOS
```bash
  https://www.python.org/downloads/
```

Start the server
```bash
  python3 -m http.server
```
